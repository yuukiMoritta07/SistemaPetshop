# 🐾 Sistema Petshop

Um sistema de gerenciamento de pet shop desenvolvido em **C**, que permite cadastrar, consultar, editar e excluir clientes e pets, com armazenamento de dados em arquivos.

---

## ✨ Funcionalidades

- ✅ **Cadastro de clientes** - Registre novos clientes do petshop
- ✅ **Listagem de clientes** - Visualize todos os clientes cadastrados
- ✅ **Busca de cliente por nome** - Localize clientes de forma rápida
- ✅ **Edição de dados de clientes** - Atualize informações dos clientes
- ✅ **Exclusão de clientes** - Remova clientes do sistema
- ✅ **Cadastro de pets** - Registre pets dos clientes
- ✅ **Listagem de pets** - Visualize todos os pets cadastrados
- ✅ **Armazenamento de dados em arquivos** - Persistência em arquivos `.txt`
- ✅ **Menu interativo no console** - Interface amigável

---

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** C
- **IDE:** Dev C++
- **Sistema:** Windows
- **Armazenamento:** Arquivos de texto (.txt)

## 🚀 Como Compilar e Executar

### No Dev C++

1. Abra o arquivo PetShop.dev no Dev C++
2. Clique em Compile (Ctrl + F9)
3. Clique em Run (Ctrl + F10)

---

## 📖 Como Usar

1. Execute o programa
2. Escolha uma opção no menu:
   - 1: Cadastrar cliente
   - 2: Listar clientes
   - 3: Buscar cliente
   - 4: Editar cliente
   - 5: Excluir cliente
   - 6: Cadastrar pet
   - 7: Listar pets
   - 0: Sair

3. Os dados são salvos automaticamente em arquivos .txt
